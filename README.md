- Open the terminal 
- Write "yacc -d phase2.y" in the terminal
- Write "flex phase2.l" in the terminal
- Write "gcc phase2.tab.c lex.yy.c -ly -ll" in the terminal
- Write "./a.out" in the terminal then enter your desired expression
